Apa itu CSS?

CSS kepanjangan dari Cascading Style Sheets. Jika html adalah bagian kerangkanya, maka CSS adalah kulitnya, yang tampak dari luar, yang mengatur tampilannya. Maksudnya CSS itu untuk mengatur tata letak, ukuran, warna, bentuk yang membuat sebuah website menjadi cantik.
