Apa Itu Javascript?

Javascript adalah suatu bahasa pemrograman yang sangat matang, artinya, dia bisa digunakan untuk memanipulasi menggunakan logika.

Lalu, apa gunanya Javascript untuk website devolpment?

Javascript berguna untuk memberikan fungsi agar website kita bisa memiliki fitur untuk pengguna. Juga bisa untuk mengubah data dalam web, animasi, dll. File Javascript berekstensi .js
