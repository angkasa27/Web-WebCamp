Menampilkan console.log

Console.log adalah sebuah function di javascript yang berfungsi untuk menampilkan log di dalam console. Console itu merupakan editor output dari suatu program, sama seperti command prompt nya windows dan terminal nya Mac.

Disini kita ingin menampilkan pesan 'Semangat Pagi!', maka kita hanya tinggal mengetikkan kode sebagai berikut = console.log('Semangat Pagi!') di javascript.
