Menampilkan Alert

Salah satu hal paling sederhana yang dapat kita gunakan dengan javascript adalah menampilkan Alert, atau biasa disebut pop-up didalam website kita. untuk melakukannya. Pertama, kita harus sambungkan file javascript yang kita buat dengan menambahkan tag di html (script src='isi directory file disini')(script).

Atau, kalau tidak ingin ribet buat file baru, bisa langsung buat tag (script) isi (script). untuk menampilkan alert kita hanya perlu menulis kode alert('Aku adalah pop-up!') di dalam javascript.
